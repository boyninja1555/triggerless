> **Note:** The generated commands will only show as valid on multiplayer. I don't know why or how but that's just how it is! Ignore the errors before pressing enter and it will work.

## Overview

`/trigger` commands are **ugly**. Why not turn them into normal ones? While imperfect, **triggerless** can do just that! Best of all, this mod is entirely client-side.

**Vanilla**  
```mcfunction
/trigger spawn
```

**Triggerless**
```mcfunction
/spawn
```
