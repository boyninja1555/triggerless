## Overview

`/trigger` commands are **ugly**. Why not turn them into normal ones? While imperfect, **triggerless** can do just that! Ignore the red command and hit enter, you will not be disappointed. Best of all, this mod is entirely client-side!

**Vanilla**  
```mcfunction
/trigger spawn
```

**Triggerless**
```mcfunction
/spawn
```

## Future Plans
